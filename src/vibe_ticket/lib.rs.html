1//! vibe-ticket - A high-performance ticket management system for developers
2//!
3//! This crate provides a comprehensive ticket management solution with features including:
4//! - Git worktree integration for parallel development
5//! - Concurrent access protection with automatic file locking
6//! - Spec-driven development with three-phase workflow
7//! - Task management within tickets
8//! - Multiple export/import formats
9//!
10//! # Concurrent Safety
11//!
12//! All operations in vibe-ticket are safe for concurrent access. The storage layer
13//! automatically handles file locking to prevent data corruption when multiple
14//! processes or users access tickets simultaneously. Lock files are created
15//! transparently and cleaned up automatically, with built-in retry logic for
16//! smooth operation under contention.
17//!
18//! # Example
19//!
20//! ```rust,ignore
21//! use vibe_ticket::storage::{FileStorage, TicketRepository};
22//! use vibe_ticket::core::Ticket;
23//!
24//! // Initialize storage
25//! let storage = FileStorage::new(".vibe-ticket");
26//!
27//! // Create a ticket (automatically locked during write)
28//! let ticket = Ticket::new("fix-bug".to_string(), "Fix login bug".to_string());
29//! storage.save(&ticket)?;
30//!
31//! // Multiple processes can safely access tickets
32//! let loaded = storage.load(&ticket.id)?;
33//! ```
