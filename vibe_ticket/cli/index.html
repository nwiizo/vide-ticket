CLI module for vibe-ticket

This module provides the command-line interface implementation for the ticket management system. It handles command parsing, user interaction, and presentation of data to the terminal.

§Structure

The CLI module is organized as follows:

  • Command parsing and argument handling
  • Interactive prompts and user input
  • Output formatting and display
  • Terminal UI components

§Example

use vibe_ticket::cli::{Cli, Commands};
use clap::Parser;

let cli = Cli::parse();
match cli.command {
    Commands::Init { .. } => {
        // Handle init command
    }
    _ => {}
}
